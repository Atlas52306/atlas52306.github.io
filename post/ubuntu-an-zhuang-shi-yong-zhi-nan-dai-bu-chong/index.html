Ubuntu 开启root远程

1.设置root密码

sudo passwd root

验证当前账户密码后设置root账户密码

2. 安装ssh server服务

sudo apt install openssh-server

3.安装成功后修改配置文件文件

ps: 若没有vim 则需要安装

vi /etc/ssh/sshd_config 
sudo apt install vim

sudo vim /etc/ssh/sshd_config
# 找到以下行并修改，如果不存在则添加：
PermitRootLogin yes

4.重启ssh服务

# 重启SSH服务：
sudo systemctl restart ssh

5. 修改dns

vi /etc/resolv.conf
